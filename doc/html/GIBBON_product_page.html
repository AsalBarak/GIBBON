Summary of the GIBBON toolbox

GIBBON is an open-source MATLAB toolbox by Kevin M. Moerman and includes an array of image and geometry visualization and processing tools and is interfaced with free open source software such as TetGen, for robust tetrahedral meshing, and FEBio for finite element analysis. The combination provides a highly flexible image-based modelling environment and enables advanced inverse finite element analysis.

Citing GIBBON

A paper is in preparation but for now please cite GIBBON using:

Kevin Mattheus Moerman. (2014). GIBBON (Hylobates Agilis). ZENODO. DOI: 10.5281/zenodo.11838

For importing the citation into reference managers go here: https://zenodo.org/record/11838#.VCW6TPmSz5N.

Alternatively the following conference paper can be cited:

Moerman, K.M., A.J. Nederveen, and C.K. Simms, “Image Based Model Construction, Boundary Condition Specification and Inverse FEA Control: A Basic MATLAB Toolkit For FEBio”, in Proceedings of the 11th International Symposium, Computer Methods in Biomechanics and Biomedical Engineering. 2013: Salt Lake City, Utah, USA.
